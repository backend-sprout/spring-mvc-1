# HttpServletResponse 기본 사용법
## HttpServletResponse 역할  
`HttpServletResponse` 는 응답과 관련된 처리를 할 수 있으며       
`응답 메시지를 생성` 및 `편의 기능을 제공`해준다.        
   
**HTTP 응답 메시지 생성**  
* HTTP 응답코드 지정 
* 헤더 생성
* 바디 생성
   
**편의 기능 제공**
* Content-Type
* 쿠키  
* Redirect
